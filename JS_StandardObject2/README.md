### ⏰Date

- Date 객체는 생성자 함수이다.
- Date 생성자 함수는 날짜와 시간을 가지는 인스턴스를 생성한다.
- 인스턴스는 현재 날짜와 시간을 나타낸다.

#### 인수를 전달하지 않는 경우

```javascript
const date = new Date();
console.log(date);
// Mon Apr 15 2024 22:45:37 GMT+0900 (대한민국 표준시)
```
